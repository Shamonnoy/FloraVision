# FloraVision

FloraVision is a beautiful, responsive React application showcasing various plant collections, including top-selling plants, trendy desk decorations, and O2-producing plants. It features a modern, clean UI built with Tailwind CSS and Vite.

## 🚀 Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Modern UI**: Styled efficiently using [Tailwind CSS](https://tailwindcss.com/).
- **Component-Based Architecture**: Modular and reusable React components (`Hero`, `TopSellingPlants`, `CustomerReview`, `BestO2Plants`, `TrendyPlants`, `Footer`).
- **Interactive Elements**: Hover effects, scalable styling, and humanized interactions.
- **Fast Build**: Fast local development server over Vite.

## 🛠️ Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed before running the project.

## 📦 Setup and Run Instructions

1. **Navigate to the project directory** (ensure you are inside the `Floravision` folder):
   ```bash
   cd Floravision
   ```

2. **Install the dependencies**:
   ```bash
   npm install
   ```

3. **Start the local development server**:
   ```bash
   npm run dev
   ```

4. **View the app**: You will see an output with a local URL (e.g., `http://localhost:5173/`). Open this URL in your web browser.

## 🏗️ Building for Production

To create an optimized production build, run:
```bash
npm run build
```
This generates a `dist` folder containing the compiled project, ready to be deployed.
